# NEURAL HARVEST – Smart Farming Assistant
## Project Overview & Technical Summary

AGROSCAN AI is a high-end, futuristic digital agriculture platform designed to empower modern farmers with AI-driven diagnostics, real-time market intelligence, and comprehensive localized farming guidance. The platform bridges the gap between traditional agricultural practices and cutting-edge digital infrastructure.

---

### 🚀 Key Features & Capabilities

1.  **Neural Diagnostic Interface**
    *   **Image Recognition**: Dedicated page for immediate visual analysis and neural mapping.
    *   **Neural Mapping**: Symptom-based correlation engine with high-confidence diagnostics.
2.  **Neural Soil Analyzer (NEW)**
    *   Chemical assessment module for NPK and pH level mapping.
    *   AI-driven crop recommendation engine based on soil health.
3.  **Government Central Portal (NEW)**
    *   Integrated gateway for Central and State subsidies/schemes (PM-Kisan, PMFBY).
    *   Real-time verification and eligibility status information.
4.  **Agri-Resource Calculator (NEW)**
    *   Precision dosage algorithms for fertilizers and seeds.
    *   Resource optimization based on land acreage.
5.  **Market Intelligence Hub**
    *   Standalone terminal for weather telemetry and scrolling market price feeds.
6.  **Neural Forum (Community)**
    *   Dedicated collaboration space for farmer-to-farmer insight exchange.
8.  **Dynamic Dual-Theme Engine (NEW)**
    *   **Mission Control (Dark Mode)**: Default ultra-glassmorphism interface for focused neural diagnostics.
    *   **Solar Clarity (Light Mode)**: High-contrast, premium light interface for outdoor use and high-visibility environments.
    *   Seamless transition with persistent user preference storage.

### 🎨 Design Philosophy: "Neural Harvest Control"

The platform utilizes a **Unique Organic-Digital Aesthetic** (Cyber-Agriculture) characterized by:
*   **Ultra-Glassmorphism**: 40px backdrop-blur and saturation-boosted panels for a high-fidelity depth effect.
*   **Genetic Flow Background**: A one-of-a-kind digital rainfall system simulating biological data streams.
*   **Neural Heartbeat**: A pulsing ambient glow that gives the platform a "living" presence.
*   **Quantum Typography**: Precise use of 'Space Grotesk' for a professional, high-tech typographic hierarchy.
*   **Sensory Nodes**: A global HUD frame with interactive nodes that react to real-time telemetry.

---

### 🛠️ Technical Stack

*   **Frontend**: React 18, Vite (Fast HMR), Tailwind CSS (Design Tokens), Framer Motion (Animations).
*   **Icons**: Lucide-React (Customized SVG implementations).
*   **Backend**: Node.js, Express, Nodemailer (Feedback Telemetry).
*   **State Management**: React Hooks (useState, useEffect) with LocalStorage persistence for language preferences.
