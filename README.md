# AGROSCAN AI – Smart Farming Assistant

A simple, powerful agriculture platform to help farmers identify diseases, learn farming practices, and view market info.

## Features
- **Bilingual Support**: English and Tamil toggle.
- **Crop Categorization**: Grains, Vegetables, and Fruits (30+ crops).
- **Disease Identification**: Symptom-based matching logic.
- **Farming Guide**: Detailed care instructions for each crop.
- **Dashboard**: Static weather and market price widgets.
- **Community**: Post and share tips with other farmers.
- **Premium UI**: Modern dark theme with emerald accents.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express (No database required).

## Setup Instructions

### 1. Backend Setup
```bash
cd backend
npm install
npm start (or node index.js)
```
The backend will run on `http://localhost:5000`.

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
The frontend will run on `http://localhost:5173`.

### 3. Usage
1. Open the browser to the frontend URL.
2. Select a category (Grains, Vegetables, Fruits).
3. Select a crop from the list.
4. View the **Farming Guide**.
5. Upload a photo (mock) and select symptoms in the **Disease Detector**.
6. Click **Check Disease** to see the diagnostic result.
7. Switch between **English** and **Tamil** using the button in the navbar.
